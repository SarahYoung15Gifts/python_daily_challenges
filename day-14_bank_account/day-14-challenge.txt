Object Oriented programming:
This is object-oriented programming because your code models a real thing (a bank account) as an object with its own data and behavior.

In your file day-14_bank_account.py:2:

You define a class blueprint
The class bank_account (at day-14_bank_account.py:2) describes what every account should have and do.

You create objects (instances) from that blueprint
